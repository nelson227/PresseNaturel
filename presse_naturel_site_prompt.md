
# Prompt de Développement – Site Web **Pressé Naturel**

## 1. Objectif du projet
Créer un site web moderne pour **Pressé Naturel**, une marque de jus naturels faits à partir de fruits et légumes frais.  
Le site doit servir principalement de :

- **Vitrine digitale** pour présenter les produits
- **Plateforme de commande personnalisable**
- **Espace de confiance** avec avis clients
- **Canal de conversion vers commande ou WhatsApp**

Le site doit transmettre les valeurs suivantes :
- Naturel
- Fraîcheur
- Santé
- Simplicité
- Modernité

Le design doit être **minimaliste, naturel et premium**.
