## 🚀 Démarrage rapide (5 minutes)

1. **Installation**:
   ```bash
   # Windows
   scripts\setup.bat
   
   # Mac/Linux
   chmod +x scripts/setup.sh && ./scripts/setup.sh
   ```

2. **Lancer le site**:
   ```bash
   npm run dev
   ```

3. **Ouvrir dans le navigateur**:
   ```
   http://localhost:3000
   ```

✅ **Vous avez un site complet et fonctionnel!**

## 🎯 Cas d'usage courants

### Je veux modifier les produits/prix
→ Éditer `lib/products.ts` et relancer le serveur

### Je veux changer les couleurs/design
→ Éditer `tailwind.config.js`

### Je veux ajouter une nouvelle page
→ Lire [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md)

### Je veux déployer le site

## 📊 Structure de fichiers clé

```
lib/products.ts           ← PRODUITS (modifier ici)
components/Header.tsx     ← MENU (ajouter liens)
app/*/page.tsx           ← PAGES (contenu)
tailwind.config.js       ← COULEURS/FONTS
public/logos/            ← IMAGES
```
